Projeto desenvolvido durante o curso

Curso: Java Completo — 60 Exercícios, POO, JavaFX, JDBC, Postman, STS, REST, Maven, SQLite, JSON, Hibernate, JPA
Instrutor: Professor Arnaldo Sousa

Autor do projeto: Rafael Silverio Americo


Este projeto em Java consome a API WeatherAPI para exibir informações meteorológicas atualizadas de qualquer cidade digitada pelo usuário.

Os dados retornados incluem temperatura atual, sensação térmica, umidade, condição do tempo, velocidade do vento e pressão atmosférica.

O objetivo deste projeto é demonstrar o uso de requisições HTTP e manipulação de dados JSON em Java, aplicando boas práticas de programação, modularização e tratamento de exceções.
